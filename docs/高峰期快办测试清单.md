# 高峰期快办测试清单

## 手机端

1. 电脑点击“手机连接”，手机扫码后应打开“手机快办”，不是电脑工作台。
2. 手机首页只显示扫码出库、快速入库、查库存、待补全四个大入口。
3. 扫码出库：拍摄标签二维码，应显示唯一设备；核对型号、容量、编号、尾号和标价后填写成交价并确认出库。
4. 快速入库：只填型号、容量、15位IMEI、收货成本，应生成库存编号并打印“待复检/特价待定”标签。
5. 查库存：输入库存编号、IMEI、尾号或型号，应返回匹配设备。
6. 待补全：快速入库的手机应出现在这里，并明确显示不能出售。
7. 晚上18点后，如仍有待补全或待打印，首页应出现关店前提醒。

## 电脑端

1. 顶部应显示“电脑工作台 · 完整处理”。
2. 首页应有“待补全”数字卡片和筛选按钮。
3. 打开待补全设备详情，补齐验机信息和大于0的销售标价，勾选“我已核对资料，允许出售”并保存。
4. 保存后设备不再显示“待补全”，并可正常出库。
5. 未完成第3步前，从手机或电脑尝试出库都应被系统拒绝。

## 忙时真实演练

1. 连续快速入库3台，确认每台库存编号唯一、IMEI不能重复。
2. 连续扫码出库3台，确认每次都必须先显示设备核对卡，再能确认出库。
3. 关店前在电脑处理“待补全”和“待打印”，处理后对应数字应归零。
4. 在每日账本检查成交价、支付方式、质保和利润是否正确。
