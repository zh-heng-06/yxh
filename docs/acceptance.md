# 正式验收标准

## 打印基准

- 失败样例：[print-failed-2026-07-16.jpg](acceptance/print-failed-2026-07-16.jpg)
- 目标样例：[label-target-30x40.png](acceptance/label-target-30x40.png)
- 打印机：Windows打印队列 `NIIMBOT B1`，端口 `USB001`，203dpi。
- 标签：30mm × 40mm，按精臣模板的出纸方向。

验收时，实体标签必须满足：中文无乱码；每个字段只打印一次；型号、系统、内存、电池、串号、特价顺序与目标一致；无对象字符串；不越界；方向一致；连续打印10张不串行、不跳纸。

## 功能验收

- [ ] 苹果手机通过移动网络登录并查看实时库存。
- [ ] 安卓手机通过移动网络登录并查看实时库存。
- [ ] 店员看不到成本和利润，老板可以看到。
- [ ] 爱思截图识别后，关键字段与人工核对一致。
- [ ] 入库生成唯一库存编号并打印目标标签。
- [ ] 手机扫码和手输编号均可出库。
- [ ] 两台设备同时出库同一库存时，只允许一次成功。
- [ ] 手机出库后，电脑与另一台手机在10秒内显示新状态。
- [ ] 盘点能列出未扫到、重复扫到和放错分区的设备。
- [ ] 预订、送修、返回、退货均保留时间线。
- [ ] 店内电脑重启后打印代理自动恢复，未完成任务不丢失。
- [ ] 云端暂时断开时允许查询本地缓存，恢复后安全同步。
- [ ] 每日备份可以实际恢复到一个独立测试环境。

任何一项未验证，均不得声称完整系统已经完成。
