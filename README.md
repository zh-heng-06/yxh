# 掌柜台 · 手机门店系统 V1

当前版本：V1.1.2。手机端支持单框语音转文字完整入库；电脑首页优先展示入库、扫码出库、今日经营概览和非零待办。

在另一台电脑继续开发时，请先阅读 [`docs/异地继续开发交接.md`](docs/异地继续开发交接.md)。该文档包含当前完成状态、聊天中已经确定的产品原则、未完成功能以及数据隐私边界。

这是正式系统目录。`D:\Codex\phone-shop-prototype` 只作为交互原型和设备联调参考，不作为生产数据源。

## 正式系统组成

- `apps/web`：苹果、安卓和电脑共同使用的PWA。
- `apps/print-agent`：店内Windows打印代理，接收云端任务并驱动 NIIMBOT B1。
- `supabase/migrations`：云端数据库、登录权限、操作日志和打印队列。
- `docs`：需求、验收标准、安全与部署文档。

## 局域网启动

首次在新电脑部署时，先安装 Python 3.12，然后在项目目录执行：

```powershell
python -m pip install -r requirements.txt
```

如需使用 NIIMBOT B1 标签打印，在 Windows 上运行 `apps/print-agent/build.cmd` 生成打印组件。

双击 `启动局域网系统.cmd`。第一次打开会要求创建门店名称、老板用户名和密码。手机与电脑连接同一个Wi-Fi后，点击电脑页面的“手机连接”扫码进入。

正式本地数据库位于 `apps/local-server/data/store.db`，采用SQLite WAL模式。不要直接编辑数据库文件。

## 免费局域网安全边界

- 本系统只允许在可信的店内私人 Wi-Fi 使用，禁止在路由器上把 TCP 4180 映射到公网，也不要连接商场公共 Wi-Fi。
- 新密码至少 10 位；登录会话最长 12 小时。旧版密码哈希会在下一次成功登录时自动升级，不需要重置原密码。
- 正式数据目录应只允许当前店主 Windows 账号、`SYSTEM` 和本机管理员访问；换电脑部署后需要重新核对文件权限。
- 交接卡是“持有链接即可查看”的顾客凭证，不要把二维码发到群聊或公开平台；系统会限制重复访问并缓存长图。
- 每天自动备份仍在同一台电脑上，只能防误操作，不能防硬盘损坏。无需购买服务也可以定期把最新备份复制到仅店主保管的 U 盘。
- 当前 HTTP 模式没有传输加密。需要跨店、远程或公网访问时，必须先升级为带正规 HTTPS 的部署，不能直接开放现有端口。

苹果手机优先使用“连接iPhone入库”：数据线连接并在手机上点“信任”后，系统会在本机读取型号、容量、系统版本、IMEI/IMEI2、序列号、电池健康和充电次数；店员只需看实机补充颜色、成色、收货成本、销售标价和库位即可入库。该功能不读取照片、通讯录、消息或账号内容，不上传互联网，也不调用AI。首次部署需安装 `requirements.txt` 中的 `pymobiledevice3`，Windows电脑还需具备 Apple Mobile Device Service（安装 Apple Devices 或 iTunes 后会提供）。

安卓、华为、荣耀、小米、OPPO、vivo、三星等手机可使用“连接安卓/华为入库”。新电脑先双击 `安装安卓连接组件.cmd` 安装 Google 官方 Android Platform Tools；手机开启开发者选项和USB调试，用数据线连接后在手机上允许本电脑调试。华为/HarmonyOS 若电脑不识别，可先安装华为官方手机助手驱动；HarmonyOS 5及以上按华为官方说明开启USB调试。系统自动读取厂商允许提供的品牌、型号、系统、容量、序列号、电池信息和IMEI，但现代安卓会限制IMEI、电池健康或充电次数，空白项必须由店员在手机设置、拨号 `*#06#` 或品牌验机页中核对，系统不会猜测。该功能同样不读取照片、通讯录、消息或账号内容，不上传互联网，也不调用AI。

“爱思截图入库”继续保留，用于深度验机资料留档、直连异常时补录以及苹果零部件序列号/匹配结果等第一版直连无法稳定提供的信息。选择爱思助手的设备详情截图后，电脑本地OCR会自动识别常用字段；截图不会上传互联网，识别过程不调用云端AI。

入库页面默认勾选“入库成功后立即打印40×30横向标签”。正式系统使用本地 `apps/print-agent/LabelPrinter.exe` 把中文和设备信息先渲染成203dpi黑白位图，再通过 Windows 队列发送给 `NIIMBOT B1`，避免浏览器打印造成乱码。未成功打印的库存卡片会显示“打印标签”；只要成功打印过一次，首页和详情便不再显示打印按钮，避免重复打印。
